Featured Projects

Named Entity Recognition for Code-Mixed Text

Python NLP CRF

Problem: Standard NER models fail on code-mixed text (Hindi-English) due to language switching and informal structure.

Solution: Implemented Conditional Random Fields with custom feature extraction including POS tags, character n-grams, and language-specific patterns. Achieved significant improvement in entity recognition accuracy.

Impact: Enables better analysis of social media data and multilingual content prevalent in Indian digital spaces.

Custom Scheduler Implementation in xv6

C Operating Systems xv6

Problem: Default round-robin scheduler in xv6 lacks support for priority-based and multi-level feedback queue scheduling.

Solution: Designed and implemented MLFQ scheduler with priority aging, system call modifications for process priority management, and comprehensive testing suite.

Impact: Improved process scheduling fairness and reduced starvation for low-priority processes while maintaining system responsiveness.

InsightGuard: Real-Time Credibility Analysis

Web Application AI SEBI Safe Space

Problem: Retail investors are vulnerable to fraudulent or misleading corporate announcements, impacting their investment decisions.

Solution: InsightGuard is a web application that leverages a hybrid AI model to provide real-time credibility analysis of corporate announcements, flagging potentially misleading information. The tool is aligned with SEBI's Safe Space initiative.

Impact: Empowers users to make more informed investment decisions by providing actionable insights and protecting them from market manipulation.
